Overview

Replace this placeholder content with the artefact's high-level summary. Use the Table view tab for structured data, the Diagram tab for SVG illustrations, and the Notes tab for free-form analysis.

When to use this template

  • Artefact would exceed ~100 lines of markdown.
  • Document benefits from navigation, side-by-side comparison, or embedded diagrams.
  • Output is intended for non-developer stakeholders.

How to use it

  1. Copy this file to 41-Reports/, 20-Requirements/, or wherever it belongs.
  2. Rename per the artefact's slug (e.g. EXPLORATION-2026-05-22-foo.html).
  3. Link it from the parent Epic / Feature via the html_artefacts: frontmatter field.
  4. Validator R15 will fail the build if the path doesn't resolve.

Strengths

  • Inline SVG, sticky table headers, tabs that work without JavaScript frameworks.
  • Dark / light mode via [data-theme], persisted to localStorage.
  • System font stack — no Google Fonts, no CDN, fully offline.
